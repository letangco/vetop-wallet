## I. Setup Environments:
### 1. Mongo env:
 You must config before init mongo database, if not the user auth cannot be add.
 
 Put all secret files to:
 ````bash
 mongo/secrets
 ````
### 2. Redis config env:
 - Config your redis at:
 ````bash
 redis/redis.conf
 ````
### 3. ENV:
All configs are in .env file
